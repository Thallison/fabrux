# Onboarding de Novos Desenvolvedores

Bem-vindo ao projeto!

## Passos Iniciais

1. Clone o repositório.
2. Instale as dependências com `composer install` e `npm install`.
3. Copie o `.env.example` para `.env` e ajuste as variáveis.
4. Gere a chave da aplicação: `php artisan key:generate`.
5. Execute as migrations: `php artisan migrate`.
6. Rode o servidor local: `php artisan serve`.

Consulte os demais arquivos em `docs/` para detalhes.
