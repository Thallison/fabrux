# Guia de Uso do GitHub Copilot

- Use prompts claros e objetivos.
- Sempre revise o código sugerido.
- Consulte a [documentação oficial](https://docs.github.com/copilot).
- Exemplos de prompts úteis:
  - "Crie um controller para autenticação usando Fortify."
  - "Gere um teste Pest para o model User."
